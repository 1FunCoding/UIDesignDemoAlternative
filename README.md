# BORAT Social Study Widget

A UI design demo for a social study coordination app — status, activities, calendar, polls, and quick decision support.

## Live Demo

**[→ Open the live app](https://1funcoding.github.io/UIDesignDemoAlternative/)**

https://1funcoding.github.io/UIDesignDemoAlternative/

## Features

- **Home** — Greeting, status, “Let’s Talk” actions, and friend list
- **Calendar** — View and add activities
- **Quick Support** — Decision support based on friends’ status and activity
- Study sessions, polls, friend profiles, and status management

## Run locally

Open `index.html` in a browser or serve the folder with any static server.
